# 环境搭建
- anaconda+pycharm
- anaconda使用
    - conda list:显示当前环境安装的包
    - conda env list: 显示安装的虚拟环境列表
    - conda create -n env_name  python=3.6
    - 激活conda的虚拟环境
        - (Linux)source activate env_name
        - (win) activate env_name
    - pip install django=1.8
